# 30 - Whack A Mole
## Introduction
This application simulates a Whack A Mole! game.

![Game](https://res.cloudinary.com/saaec/image/upload/v1612793225/whackamole_ejapcr.jpg)

## Functionality
* Start (15 seconds)
* Whack a mole appearing on its hole!
