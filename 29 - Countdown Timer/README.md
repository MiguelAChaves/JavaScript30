# 29 - Countdown Timer
## Introduction
This application simulates a custom countdown timer.

![Timer](https://res.cloudinary.com/saaec/image/upload/v1612793345/countdown_cnsujb.jpg)

## Functionality
* 20 Seconds Tiner
* 5 Minutes Timer
* 15 Minutes Timer
* 20 Minutes Timer
* 1 Hour Timer
* Set number of minutes
